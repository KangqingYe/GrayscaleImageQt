# GrayscaleImageQt
基于Qt C++的灰度图像处理软件
## 功能
- 读入、显示、保存、另存图片
- 显示灰度直方图
- 平滑操作：高斯滤波、均值滤波、中值滤波
- 锐化操作：Sobel滤波x方向、Sobel滤波y方向、Laplace滤波
- 灰度变换：设置窗位窗宽、gamma变换
- 几何变换：上下翻转、左右翻转、顺时针旋转90度、逆时针旋转90度
- 形态学：腐蚀、膨胀、开运算、闭运算、提取边缘
- 分割：手动阈值、Otsu阈值分割
